# AncientScrolls

A completely server-side mod that adds a lot of content to vanilla SMP MC such as a Terraria-inspired equipment system.

This mod was programmed entirely in the SpigotMC API and must be run on Spigot or Spigot-forked (PaperMC) servers. It does not require modding APIs such as Forge or Fabric to run.

Despite having official releases, this mod is still somewhat experimental as I, the original creator, am the only one developing it. Expect (and please report) bugs if you decide to use this plugin on your server.

## Features

- Scrolls
- Armor trim bonuses
- Flasks
- More features are to be added in later releases.

## Scrolls

- Items that apply a certain bonus or enhancement to a player when equipped
- Chance to generate as treasure in loot containers or as mob drops
- Player can equip scrolls by right-clicking with the item or moving it into their <code>/as</code> inventory.
